Where do the new steps go?

The spec adds 5 new steps (Q1–Q5). Three ways to slot them into the existing 9-step wizard.

A

Professional-first

New steps up front, then existing lifestyle steps. Matches the spec's intended order.

NEW Q1 Professional role
NEW Q2 Current priority (JTBD)
NEW Q3 Financial mindset
NEW Q4 Values fingerprint
NEW Q5 Passions
── existing: marital status, kids, movies, food, allergies, location, shops, travel, freeform
B

Lifestyle-first (additive)

Keep existing steps in place, append new ones after. Minimal disruption, safest for tested flows.

── existing: marital status, kids, movies, food, allergies, location, shops, travel, freeform
NEW Q1 Professional role
NEW Q2 Current priority (JTBD)
NEW Q3 Financial mindset
NEW Q4 Values fingerprint
NEW Q5 Passions
C

Grouped by theme

Re-order into logical groups: identity → professional → values → lifestyle. Feels most coherent but requires reordering existing steps.

Identity: marital status, kids, location
NEW Professional: Q1 role, Q2 priority, Q3 financial mindset
NEW Values: Q4 fingerprint, Q5 passions
Lifestyle: movies, food, allergies, shops, travel, freeform
Note on total step count: Adding 5 new steps brings the wizard to 14 steps. Option C also removes the standalone freeform step (Q2 replaces it as the intent capture). That lands at 13 steps. Any of these can be chunked into progress sections (e.g. "About you · Work · Interests") to reduce perceived length.
